Buatkan website yang menampilkan post dari user.
user memasukan link youtube yang menjadi embeed di website yang bisa di play.
post itu memiliki kategori sebagai berikut


-Tajwid
-Surat Pendek
-Doa doa harian
-Surat Pilihan
-Hadis dan adab
-Tata cara sholat
-Tata cara berdoa
 
Di home page akan menampilkan pilihan kategori tersebut
setelah di klik akan muncul semua postingan dari user2 berdasarkan kategori tersebut

User dapat login dengan username dan password. 
user juga dapat register

tech spec:
nextjs, shadcn, db (sqlite untuk development) dan turso (untuk production), eslint, better auth, strict typescript, tailwind,
drizzle
gunakan context7 mcp untuk menggunakan latest version

Implementation:
Untuk implementasi dipisah menjadi 2 phase: phase pertama adalah frontend menggunakan dummy data. lalu phase kedua melakukan integrasi frontend ke database.
